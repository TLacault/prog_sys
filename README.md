# Programmation Système - PG204

## Utilisation des TD

Le dossier **`00-template`** contient les fichiers **`utils`** update avec tout les headers nécessaires pour compiler les TP.
Le makefile est aussi mis à jour pour compiler avec les bonnes options (`-pthread`, etc).

Chaque TP contient le **`sujet.txt`** et une **fiche markdown** expliquant les concepts clés.
Pour refaire et compiler les exos, vous pouvez copier les fichiers **`utils`** et **`Makefile`** du dossier `00-template` dans le dossier du TP.
Ou alors juste dev dans le dossier **`00-template`** directement.

## Fiches

Les fiches markdown sont dans le dossiers de leur TP respectif.
Leur version compilé en pdf et imprimable est dans le dossier **`fiches/`**.
